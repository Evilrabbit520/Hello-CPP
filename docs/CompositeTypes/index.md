# 第 4 章 复合类型

!!! abstract "本章内容概述"

    - 如何创建和使用数组；
    - 如何创建和使用 C 风格的字符串；
    - 如何创建和使用 `string` 类的字符串；
    - 如何混合输入字符串和数字；
    - 如何使用 `getline()` 和 `get()` 方法来读取字符串；
    - 如何创建和使用结构；
    - 如何创建和使用共用体；
    - 如何创建和使用枚举；
    - 如何创建和使用指针；
    - 如何管理动态内存；
    - 如何创建和使用动态数组；
    - 如何创建动态结构；
    - 了解自动存储、静态存储和动态存储。

嘿，恭喜你，一路披荆斩棘的离开了数据操作的奇妙森林，很高兴看到你用基本数据类型和各种运算符玩得那叫一个不亦乐乎，就像熟练的厨师在厨房里摆弄着锅碗瓢盆，做出了一道道简单却美味的数据“菜肴”。

但请不要以为编程的冒险就到此为止啦！现在，咱们要一脚迈进第 4 章复合类型这个神秘又刺激的全新世界。如果说第 3 章的基本数据类型是一个个孤零零的小士兵，那这章的复合类型就是由它们组成的强大军团，有能批量收纳数据的数组“大仓库”，有像超级导航一样精准定位的指针，还有和变量亲密无间的引用“好兄弟”。

更厉害的是，这里还有 `new` 和 `delete` 这对神奇搭档，能像魔法师一样在程序运行时随意召唤和遣散内存空间，打造出各种动态结构和数组。这就好比咱们从操控小士兵升级成了指挥千军万马的大将军，能构建出更加宏伟壮观的编程城堡！

准备好了吗？让咱们一起撸起袖子，在复合类型的世界里大干一场，创造出属于咱们的编程传奇！
